# Calculator

A simple dark-themed calculator created using HTML, CSS, and JavaScript.  
It performs basic arithmetic operations with a clean and responsive design.

## Features
- Addition, subtraction, multiplication, division, and percentage
- Decimal number support
- Clear and backspace functions
- Dark mode UI
- Responsive layout

## Technologies Used
- HTML
- CSS
- JavaScript

## Project Files
index.html – main structure
style.css – design and layout
script.js – calculator logic
